小白天气(基于CoolWeather修改)
===========

本项目更新中……

小白天气是一款基于Android端开源的天气预报软件，具备查看全国的省市县、查询任意城市天气、自由切换城市、手动更新天气、后台自动更新天气等功能。

小白天气基于[郭霖](https://blog.csdn.net/guolin_blog)老师的《第一行代码》[CoolWeather ](https://github.com/guolindev/coolweather)

小白天气中的天气数据由和风天气提供，背景图片由必应提供，代码遵循Apache v2 License开源协议。本软件主要作为学习和交流使用。
